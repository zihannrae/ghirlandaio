**<h1 align="center">Apa Perbedaan Open Source dan Closed Source</h1>**

### Pendahuluan
Software atau dalam KBBI yang ditulis perangkat lunak adalah program atau aplikasi yang diisikan ke dalam memori internal komputer. Software disini adalah perangkat yang tidak memiliki bentuk fisik, melainkan sebuah bentuk kode pengoperasian yang dilakukan melalui mesin komputer. Selain itu software juga berisikan data yang disimpan atau diprogram yang memiliki berbagai fungsi. Perkembangan software ini melahirkan berbagai jenis sistem dangan berbagai karakter, salah satunya tentang keterbukaan kode sumbernya yaitu menjadi open source dan closed source. Pembagian ini penting untuk dipahami karena berkaitan dengan bagaimana software digunakan, dikembangkan serta bagaimana keterbukaannya kepada pengguna.
### Pembahasan
#### Open Source
Software open source adalah jenis software yang memungkinnya penggunanya untuk mengakses, mengubah ataupun membagikan kode sumber secara terbuka dan bebas. Kode sumber biasanya berisikan sekumpulan instruksi yang digunakan untuk pengembangan sebuah progaram melalui komputer. karena bersifat lebih terbuka, ini membuat kode yang ada bisa di pelajari oleh siapa saja dan bebas untuk menggunakannya. Tidak hanya itu, pengguna juga dapat memodifikasinya dengan sesuka hati, bahkan juga bisa mengembangkan kode itu. Karna hal ini, potensi dari open source ini bisa sangat luas karna bisa dikembangkan oleh siapa saja sesuai dengan kebutuhan yang ada. Open source biasanya dimiliki oleh komunitas pengembang yang aktif dalam pengembangan dan terlibat langsung diprogram itu. Disini komunitas bertanggung jawab untuk mengembangkan, memperbarui, menjaga dan menyelesaikan berbagai masalah pada software tersebut. 
#### Closed Source
Software closed source adalah jenis software yabg tidak memberikan akses kepada para pengguna untuk melihat, mengubah, ataupun membagikan kode sumbernya secara bebas. Kode sumber pada software ini bersifat tertutup dan hanya dapat diakses oleh pihak pengembang atau perusahaan pribadi yang membuatnya. Karena sifatnya yang terbatas, pengguna hanya dapat menggunakan software tanpa bisa melihat kode sumbernya karena aturan dan lisensi yang ditetapkan oleh pribadi perusahaan. Pengembangan, perbaikan, dan pembaruan sepenuhnya merupakan tanggung jawab dan dikendalikan oleh perusahaan yang mengembangkannya, sehingga pengguna tidak memiliki peran untuk mengembangkannya. Untuk keamanan, closed source biasanya lebih ketat dan dalam pengujian yang lebih terkontrol, karena sistemnya dikelola oleh internal pihak pengembang dan agar tidak bocor ke luar dari pihak pengembang.
#### Perbedaan dan contoh open source dan closed source
Perbedaan antara software open source dan closed source terletak pada keterbukaan kode sumbernya. Open source memiliki kode yang terbuka sehingga dapat diakses dan dimodifikasi penggunanya. Closed source bersifat tertutup dan hanya dapat diakses sesuai dengan aturan pengembang yang mengembangkannya. Dari segi biaya, open source biasanya gratis, sementara closed source berbayar, selain itu open source juga dikembangkan oleh komunitas dan dikembangkan tergantung dengan kebutuhannya sedangkan closed source di kendalikan dan dikembangkan oleh pribadi, biasanya perusahaan. Adapun contoh dari open source yaitu ada Linux, Apache HTTP Server, dan MySQL dan dari closed source ada Microsoft Office, Adobe Photoshop, dan AutoCAD.
